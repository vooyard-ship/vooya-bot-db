## 📋 사용 규칙
- **재주문 시**: 주문 이력에 새 행 추가 → 체크박스는 `[ ]`으로 유지 (코드가 최신 날짜 자동 선택)
- **소진 시**: 해당 항목 체크박스를 `[v]`로 변경
- **재주문 + 이전 것 소진**: 체크박스는 `[ ]` 유지 + 주문 이력에 새 행만 추가 (행 두 개 만들지 않아도 됨)
- **주문 이력**: 쌓이는 대로 유지 (용량 무시해도 됨, 최신 날짜 기준으로만 계산됨)
