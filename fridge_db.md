## 📋 사용 규칙
- **재입고 시**: 입고 이력에 새 행 추가 → 체크박스는 `[ ]`으로 유지 (코드가 최신 날짜 자동 선택)
- **소진 시**: 해당 항목 체크박스를 `[v]`로 변경
- **재입고 + 이전 것 소진**: 체크박스는 `[ ]` 유지 + 입고 이력에 새 행만 추가 (행 두 개 만들지 않아도 됨)
- 주문 이력 날짜**: 온라인 주문은 주문일 입력 (코드가 +1일 = 배송일로 자동 계산) / 시장·선물·직접조달은 받은 날 입력
- **요리한 음식**: 만든 날 추가, 소진 시 `[v]` → 유통기한 알림 대상에서 제외됨 (입고 이력에 없으므로)
- **상시 보유 조미료**: 소진될 때만 `[v]` 처리 (유통기한 추적 불필요)
